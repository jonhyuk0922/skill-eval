# 채점표 템플릿 (출력 형식의 단일 출처)

평가 결과는 아래 형식 그대로 낸다. **이 파일이 출력 형식의 유일한 원본**이다 —
`SKILL.md` 와 `scripts/collect.py` 는 여기를 가리키고, collect.py 는 이 블록을 읽어 빈 채점표를 찍는다.
`<…>` 는 채워 넣는 자리다.

```
## 검수 결과 — <스킬 이름>   합계 <N>/10

| 축 | 판정 | 근거 (본문 한 줄) |
|---|---|---|
| ① 명세 또렷함 | <합격/부분/미달> | "<본문에서 인용>" |
| ② 검증 루프 〔사람〕 | <합격/부분/미달> | "<본문 인용, 없으면 '돌려본 흔적 없음'>" |
| ③ 진짜 업무성 〔사람〕 | <합격/부분/미달> | "<본문 인용, 없으면 '반복 업무 근거 없음'>" |
| ④ 동료 이식성 | <합격/부분/미달> | "<본문에서 인용>" |
| ⑤ 임팩트 | <합격/부분/미달> | "<본문에서 인용>" |

🎯 오늘 고칠 자리: <가장 점수 낮은 축 이름>
🩺 걸린 결함: <뭉뚱그린 명세 / 한 방에 끝 / toy 예제 중 해당, 없으면 "없음">
💊 한 줄 처방: <가장 짧은 축을 올리는 복붙용 프롬프트 한 줄>
```

**〔사람〕 = 기계 휴리스틱이 못 가리는 축.** collect.py 의 참고 숫자가 조용해도
본문을 직접 읽고 사람이 판정한다. ②③은 자동 신호를 근거 칸에 쓰지 않는다.

## 작성 규칙

- 근거 칸은 **반드시 대상 SKILL.md 본문에서 따온 실제 문장**. 추측 금지.
- ②③은 자동 숫자를 근거로 쓰지 말 것 — 본문 문장을 인용하거나, 없으면 "…흔적 없음"이라 사람이 표시한다.
- 처방은 한 줄, 복붙 가능, 대상 스킬에 바로 끼울 만큼 구체적으로. **"가장 짧은 축" 하나에 대해서만.**
